# 🧮 Calculadora Simples

Este projeto é uma **calculadora básica desenvolvida com HTML, CSS e JavaScript**, com o objetivo de praticar conceitos fundamentais de desenvolvimento front-end.

## 📌 Funcionalidades

* Operações matemáticas básicas:

  * Adição (+)
  * Subtração (−)
  * Multiplicação (×)
  * Divisão (÷)
* Limpar visor (C)
* Interface simples e intuitiva

## 🚀 Tecnologias utilizadas

* **HTML5** – Estrutura da aplicação
* **CSS3** – Estilização e layout
* **JavaScript** – Lógica e interatividade

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com foco em:

* Praticar manipulação do DOM
* Trabalhar com eventos (clicks de botões)
* Reforçar lógica de programação
* Melhorar organização de código em projetos pequenos

## 📂 Estrutura do projeto

```
📁 calculadora/
 ├── index.html
 ├── style.css
 └── script.js
```

## 💡 Aprendizados

Durante o desenvolvimento, foram reforçados conceitos como:

* Seleção de elementos com JavaScript
* Funções e operadores matemáticos
* Manipulação de strings e números
* Organização de código
